# EHDA APP
Ehda Android Application (React Native)


# INSTALLATION

```
npm install -g yarn

yarn global add react-native-cli
yarn install
```

### ANDROID

1) config a proxy at port 1081 then
2) run an emulator
```
react-native run-android
```

### IOS
disable global proxy on mac-ox then:
```
react-native run-ios

```

# test

```
yarn test-watch

```
